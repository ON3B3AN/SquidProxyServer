Project Background

Our project came to light from the idea that organizations can make use of and implement a proxy service which allows them to analyze their web traffic and blacklist sites for any number of reasons. A proxy server is a machine which sits between your computers on a Local Area Network and the larger Wide Area Network. Proxies have many usages, but ours is focused on traffic monitoring and content filtering. Our custom service is an implementation of the popular Squid Proxy Server, with several features on top of it to customize it. We are utilizing SSL Bump to monitor all web traffic of our clients connected to the proxy server. The web traffic is stored by Squid and interpreted by another application called Squid Analyzer, which displays them in easy-to-read charts, graphs, and tables.
